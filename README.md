## Installation

To run the project firstly ensure that Python and pip is correctly installed and it's path is added in the environment variables.

Check using the below command in terminal

```bash
  python --version
  pip --version
```
Also install the pygame module and the moviepy module of python initially in your device.
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/VaibhavG99/pygame-project-CAR-RACE-2D.git
```

Go to the project directory
open the "main code.py" fie in your VS code
Build and Run the application.
The application will run shortly.
